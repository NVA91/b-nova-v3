# Installationshinweise & Sicherheit

## GPU-Schutz aktiviert

Dieses Setup erlaubt **ausschließlich** die Verwendung der NVIDIA RTX 5060 Ti.

⚠️ Die RTX 3080 oder andere GPUs werden bewusst blockiert, um Schäden zu vermeiden.

## AWX / Controller Umgebung

Das Verzeichnis `controller/` enthält das Setup-Playbook für eine isolierte AWX/Tower-Testumgebung.
